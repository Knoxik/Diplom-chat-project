Сайт для общения двух людей. Чат использует end-to-end шифрование и 2048-bit RSA ключ.<br>
Проект на [https://dashboard.heroku.com/apps/diplom-chat-project](Heroku).
## Стек технологий

`- HTML5`
`- CSS3`
`- VueJS(ES6+)`
`- Socket.io`
`- Node.js`
`- Express`
`- RSA encryption`
`- npm`

## Запуск

`- npm start`
